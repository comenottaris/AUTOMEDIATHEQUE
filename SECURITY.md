# Politique de Sécurité / Security Policy

##  Français

### À propos de ce projet

Automédiathèque est un site statique (HTML, CSS, JavaScript) sans backend ni base de données. Les données sont stockées dans des fichiers JSON publics. Le projet est hébergé sur GitHub Pages.

### Périmètre de sécurité

Ce projet ne traite **aucune donnée sensible** :
- Pas d'authentification utilisateur
- Pas de base de données
- Pas de traitement côté serveur
- Pas de collecte de données personnelles

### Signaler un problème

Si vous identifiez un problème de sécurité (XSS, liens malveillants, etc.), vous pouvez :

1. **Ouvrir une issue** sur ce repository
2. **Contacter directement** : [ouvrir une issue privée](https://github.com/comenottaris/AUTOMEDIATHEQUE/issues/new)

### Vulnérabilités hors périmètre

- Vulnérabilités dans les navigateurs
- Vulnérabilités dans GitHub Pages
- Attaques nécessitant un accès physique

##  English

### About this project

Automédiathèque is a static website (HTML, CSS, JavaScript) with no backend or database. Data is stored in public JSON files. The project is hosted on GitHub Pages.

### Security scope

This project handles **no sensitive data**:
- No user authentication
- No database
- No server-side processing
- No personal data collection

### Reporting an issue

If you identify a security issue (XSS, malicious links, etc.), you can:

1. **Open an issue** on this repository
2. **Contact directly**: [open a new issue](https://github.com/comenottaris/AUTOMEDIATHEQUE/issues/new)

### Out of scope

- Browser vulnerabilities
- GitHub Pages vulnerabilities
- Attacks requiring physical access
